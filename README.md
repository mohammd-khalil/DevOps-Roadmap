# DevOps Roadmap

A practical roadmap for becoming a DevOps Engineer with free resources and hands-on projects.

# - Phase 1 — Linux

## Learn

* Linux Basics
* File System
* Users & Permissions
* Process Management
* Networking Commands
* Shell Scripting

## Resources

* Linux Full Course (Playlist)
  https://www.youtube.com/watch?v=oD5Y4Gzr6vw&list=PLy1Fx2HfcmWBpD_PI4AQpjeDK5-5q6TG7

* Linux Crash Course
  https://www.youtube.com/watch?v=WMy3OzvBWc0

* Build your own Linux lab using VirtualBox or WSL.
* Create Bash scripts for automation.
* Learn SSH, cron jobs, permissions, and package managers.
